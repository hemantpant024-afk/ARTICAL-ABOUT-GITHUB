# _GitHub Guide_

This is a github guide which tells you about the github basic commmands

- `git clone <repo-url>` - To clone the repo
- `git status` - To check the track & untracked files.
- `git add ` - To add folder in repo
- `git commit` - To conform
-  `git push` - push the folder in remote 
-  `git pull` - pull for remove folder from remote
